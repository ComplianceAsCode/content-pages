Definition of Firefox Common User Security Policy for firefox

based on jodehnal's bachelor thesis on creating a SCAP profile for common users of Fedora workstation - link will be added after publication

1.0: Firefox Hardening

Description: Secure configuration of Mozilla Firefox.

Levels:
  • default

Automated: yes

Selections:
  • firefox_policy-autoplay_video: Firefox autoplay must be disabled.
  • firefox_policy-content_blocker: Ensure the Content Blocker uBlock Origin is Installed
  • firefox_policy-cryptomining: Enabled Firefox Cryptomining protection
  • firefox_policy-enhanced_tracking: Enabled Firefox Enhanced Tracking Protection
  • firefox_policy-fingerprinting_protection: Enabled Firefox Fingerprinting Protection
  • firefox_policy-javascript_window_changes: Disable JavaScript's Raise Or Lower Windows Capability
  • firefox_policy-javascript_window_resizing: Disable JavaScript's Moving Or Resizing Windows Capability
  • firefox_policy-pop-up_windows: Enable Firefox Pop-up Blocker
  • firefox_policy-ssl_minimum_version: Firefox must be configured to allow only TLS 1.2 or above.
