Uninstall pigz Package

package_pigz_removed

Description

The pigz package can be removed with the following command: 

$ sudo dnf remove pigz

Rationale

Binaries shipped in pigz package in AlmaLinux OS 9 have not been compiled using recommended compiler flags. The binaries are compiled without sufficient stack protection and its address space layout randomization (ASLR) is weak.
