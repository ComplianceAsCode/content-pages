Rule details

Disable Cockpit Management Server

service_cockpit_disabled

Description

The Cockpit Management Server (cockpit) provides a web based login and management framework. The cockpit service can be disabled with the following command: 

$ sudo systemctl mask --now cockpit.service

Rationale

Cockpit provides a form of remote login.
